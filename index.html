Getting Started

  1. Clone the repo: git clone https://github.com/s0m3on35/satellite-defense-toolkit
  2. Install Python dependencies: pip install -r requirements.txt
  3. Launch: python3 webgui_web/app.py
  4. Access dashboard at http://localhost:5000
